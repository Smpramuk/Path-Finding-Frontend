Project Description: The goal of this project is to help visualize and compare a few pathfinding algorithms. The algorithms consist of Depth-first search (DFS), Breadth-first search (BFS), the A* algorithm, and Dijkstra's algorithm. Click any of these algorithm buttons to begin searching for a path between the start and end points. Click and drag to create walls or weighted nodes depending on which is toggled. Click the weighted nodes and walls button to toggle between them. Add new points via the new points button. Clear the grid (except for the start and end points) with the reset grid button. This page was created using JavaScript, HTML, and CSS.
